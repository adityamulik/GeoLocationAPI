GeoLocation API (File Upload)

This application is for getting the co-ordinates of a location by uploading a excel file with any particular address. The co-ordinates are fetched using MapQuest's REST API.

Rules to follow:

1. Please upload a excel file with .csv extension only.
2. Please add column headers as address, latitude & longitude. (**Case-sensitive)
3. Please add any number of addresses of your choice in the first column only.
4. All other columns should be blank for this web application to work.
5. Click Upload the file, a output file will be generated in 10-120 seconds depending on the file size.
